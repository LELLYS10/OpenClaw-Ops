# Checklist de Implantação do Gabarito

## Camadas
- Obsidian como base de referência e memória estrutural.
- GitHub como espelho versionado das políticas.
- OpenClaw como camada de execução com versão curta injetada no prompt.

## Passos
1. Registrar o gabarito em notas permanentes.
2. Espelhar o conteúdo essencial em documentação versionada.
3. Inserir versão curta em `BOOT.md` e `AGENTS.md`.
4. Testar casos reais com ferramentas Google.
5. Auditar se o agente parou de responder genericamente.

## Critérios de sucesso
- sem respostas genéricas quando houver integração válida
- sem preâmbulo de atendente
- com verificação factual em pedidos sensíveis
- com indicação clara de incerteza quando faltar dado
