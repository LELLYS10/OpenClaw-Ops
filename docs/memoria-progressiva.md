# Memoria Progressiva

## Objetivo
Fazer o BOBY esquecer menos sem transformar memoria em bagunca.

## Camadas
1. `MEMORY.md`
- fatos duraveis
- preferencias estaveis
- regras de tratamento
- contexto persistente de pessoas e sistemas

2. `memory/YYYY-MM-DD.md`
- diario operacional
- pedido
- acao
- pendencia
- proximo passo

3. `memory/people/`
- fatos duraveis por pessoa

4. `memory/projects/`
- estado resumido por projeto ou sistema

5. `memory/checklists/`
- rotinas fixas de boot, revisao e consolidacao

## Regra de gravacao
- fato importante do dia vai para o arquivo diario
- fato recorrente ou duravel sobe para `MEMORY.md`
- fato ligado a uma pessoa sobe tambem para `memory/people/`
- fato ligado a um sistema sobe tambem para `memory/projects/`

## Regra de qualidade
- nao prometer memoria sem gravar
- nao encher `MEMORY.md` com fatos passageiros
- nao deixar memoria so no modelo
- consolidar texto curto, objetivo e reutilizavel

## Fechamento de tarefa
Ao fim de tarefa relevante, registrar:
- pedido
- o que foi feito
- pendencia
- proximo passo
