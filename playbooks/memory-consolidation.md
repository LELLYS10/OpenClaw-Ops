# Consolidacao de Memoria

## Quando usar
- ao final de tarefa relevante
- quando o MESTRE mandar `registre na memoria`
- quando surgir preferencia duravel
- quando houver novo estado estrutural do OpenClaw

## Fluxo
1. Registrar no diario do dia em `memory/YYYY-MM-DD.md`
2. Se o fato for duravel, consolidar em `MEMORY.md`
3. Se envolver pessoa, registrar em `memory/people/`
4. Se envolver sistema ou projeto, registrar em `memory/projects/`

## Diario do dia
Usar formato curto:
- Pedido:
- Ação:
- Pendência:
- Próximo passo:

## Duravel
Vai para `MEMORY.md` apenas se for:
- preferencia estavel
- regra de tratamento
- fato pessoal importante
- estado estrutural de sistema

## Regra
- memoria progressiva sem consolidacao vira ruido
- consolidar pouco, mas consolidar certo
