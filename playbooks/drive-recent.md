# Drive Recent

## Objetivo
Listar arquivos recentes do Google Drive com dados reais.

## Comando
`python3 /data/.openclaw/workspace/google_bridge.py drive-recent`

## Resposta esperada
- nome do arquivo
- tipo
- data de modificação
- link quando útil

## Regra
Se a integração local funcionar, não responder com tutorial de configuração.
