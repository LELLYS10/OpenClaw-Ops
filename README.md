# OpenClaw Ops

Repositório dedicado à operação do BOBY/OpenClaw.

## Estrutura
- `docs/`: políticas, gabarito e estado atual
- `playbooks/`: rotinas operacionais versionáveis
- `prompts/`: regras curtas de comportamento
- `runtime/`: espelho dos arquivos críticos da VPS
- `obsidian/`: recorte do vault operacional do BOBY

## Documentos-chave
- `docs/arquitetura-operacional.md`
- `docs/checklist-prontidao.md`
- `docs/matriz-validacao.md`

## Objetivo
- separar OpenClaw da bagunça do Desktop
- versionar o que é publicável
- alinhar VPS, Obsidian e documentação

## Regras
- não subir credenciais
- não misturar projetos CredPlus ou arquivos aleatórios aqui
- manter este repositório focado em operação e qualidade do BOBY
